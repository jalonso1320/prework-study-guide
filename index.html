HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • Externl CSS style sheet allows developers to keep all their CSS rules in a seperate file.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • Link element ceates the connection to the external file.
  • Rel attribute specifies thet relationship between the current document and the linked document.
  • Herf attreibute specifies the location (URL) of the external resource.
  • Img selector assigns property values to any image element i the markup.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds the changed files in our working directory to the staged area of git
  • git commit -m: commits changes to our repository
  • git pull origin main: pulls the base branch
  • git push origin feature/branch-name: pushes up all of our changes
